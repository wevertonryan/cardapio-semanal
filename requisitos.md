Ideia: 
- Sistema que permite eu criar o meu cardápio semanal e ver se está batendo com as minhas necessidades, e dizer se é um cardápio saudavel ou não

Requisitos:
- Baixar automaticamente os dados atualizados
- Banco de receitas
- Permita a pesquisa das receitas por ingredientes
- Permita a inserção dos ingredientes que eu tenho em casa
- Permita realizar o calculo do que eu preciso com base no meu perfil
- Dashborad dizendo a variedade de nutrientes na semana e qual saúdavel está, falando também quantos nutrientes de cada esse cardápio vai proporcionar, e quais vão ficar faltando
- Posteriormente incluir calculos baseados em tabela de combinação alimentar tbm
- Cada refeição vai incluir seus valores nutricionais, incluindo também se está de acordo com a tabela de combinação alimentar
- Funcione offline

Import:
- Baixar dados
- Descompactar
- Se possivel armazenar ele convertido como um arquivo
- Importar apenas os dados que importam para o IndexDB
